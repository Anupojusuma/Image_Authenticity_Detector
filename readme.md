# Image Authenticity Detector

A forensic-based system that analyzes images to estimate authenticity using metadata, compression analysis, and noise patterns.

## Features

- EXIF metadata analysis
- Editing software detection
- SHA-256 image integrity hashing
- Error Level Analysis (ELA)
- Noise Level Analysis (NLA)
- Authenticity scoring system

## Tech Stack

- Python
- Streamlit
- OpenCV
- NumPy
- Pillow
- EXIFRead

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
